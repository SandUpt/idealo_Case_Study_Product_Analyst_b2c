# idealo_Case_Study_Product_Analyst_b2c
idealo User Journey Analysis Project
This contains the analysis for the idealo Product Analyst case study. It includes data exploration, user flow analysis, SQL queries and the final presentation with feature recommendations.

Main Files:
case_study.ipynb - Jupyter notebook with all steps: data cleaning, user journey analysis, traffic source breakdown, conversion metrics and SQL queries.
Case_Study.pptx - PowerPoint presentation summarizing findings, key metrics, visualizations and the proposed homepage feature improvement.
